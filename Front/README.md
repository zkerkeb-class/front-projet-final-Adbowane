**Application**
Nom Prénom: Abdoulaye Wane 

**Liste des fonctionnalités :**

- Authentification (connexion, inscription, gestion de session)
- Tableau de bord avec statistiques (membres actifs, finances, activités, taux de participation…)
- Gestion des membres :
  - Liste des membres
  - Ajout, modification, suppression de membres
  - Téléversement et gestion de documents d’adhésion
  - Changement de statut (actif, suspendu, expiré)
- Gestion des activités :
  - Création, modification, suppression d’activités
  - Affichage des activités à venir et passées
- Gestion des cotisations :
  - Liste des cotisations
  - Ajout, modification, suppression de cotisations
  - Filtrage et recherche
- Gestion documentaire :
  - Liste des documents (PDF, Word, Excel, images…)
  - Filtrage par catégorie et recherche
  - Téléchargement de documents
  - Ajout de documents
  - Suppression de documents
- Gestion du profil utilisateur
- Paramètres de l’application :
  - Exportation et importation des données (CSV/JSON)
  - Personnalisation (thème, notifications, langue…)
- Centre d’aide et FAQ
- Contact/support (formulaire de contact)
- Administration MongoDB :
  - Visualisation des logs d’activité
  - Visualisation des notifications système
- Thème sombre et le thème clair

**Use your preferred IDE**

If you want to work locally using your own IDE, you can clone this repo and push changes. Pushed changes will also be reflected in Lovable.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
